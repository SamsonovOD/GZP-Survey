Сайт состоит из заглавия, основной панели, боковой панели и подвала. Заглавие и подвал содержат только текст.

Основная панель отображает или текущий опрос, информационное сообщение связанной с состоянием опросов, или профиль пользователя.

Боковая панель содержит форму авторизации и открывающуюся форму регистрации.

При регистрации введенные данные сохраняются в БД. Пароль зашифрован через SHA258. Подтверждение пароля проверяет сходство двух полей для пароля. Для проверки стенда опционально можно установить режим администратора для пользователя.

При входе данные сверяются с данными в БД. При соответствии, данные сохраняются в память сервера SESSION на короткое время. При выставлении метки «запомнить», данные сохранятся на несколько часов.

Опрос отображается только для авторизованных пользователей. Опрос представляет собой форму разделённую на вопросы. Вопросы бывают следующих типов:
* Выбор одного ответа.
* Выбор нескольких ответов.
* Ранжирование ответов.
* Свободный ответ.
При отправке, ответы сохраняются в БД за каждого пользователя. Если на текущий опрос есть ответы текущего пользователя, опрос не отобразится.

В профиле отображены информация о пользователе и предыдущие ответы пользователя. Если пользователь имеет функции администратора, то доступны следующие функции:
* Смена текущего опроса в БД (При этом если станица открыта на опросе не соответствующей текущей, появится предупреждение и опрос закроется.)
* Список ответов всех пользователей.
* Гистограммная статистика ответов.
* Редактор, в котором можно добавлять и удалять опросы, вопросы, указать тип вопроса, и ответы.

Сайт загружен и установлен на бесплатный хост. Работоспособность через интернет и с телефона проверена.
